Games Collection Documentation

Here you will find information on using the games collection api.



ADDING GAMES

POST /api/v1/games - Add some new games to our database!

When adding games, please use this as your body to ensure uniformity in our database.

(Fill in as many fields as you can - variables show desired types of data)

    {
        "username": "String",
        "name": "String",
        "rating": Float,  (Fun rating, 0.0-10.0 scale)
        "minplayers": Int,
        "maxplayers": Int,
        "minplaytime": Int,
        "maxplaytime": Int,
        "yearpublished": "String",
        "ranking": Int,   (Game ranking overall, 1 being the highest)
        "numberowned": Int,
        "type": "String"
    }


GETTING GAMES

GET /api/v1/games - Get all the games from our database!
GET /api/v1/games/<id> - Get a specific game from our database!

DELETING GAMES

DELETE /api/v1/games/<id> - Delete a game from our database!
